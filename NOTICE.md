# Notice

This repository contains crawler, translation, and static-site code only. It does **not** bundle a copy of any source website.

When you run it, it creates a personal local text mirror from pages made publicly available at the configured source URL. Source content remains attributable to the original publisher and may be protected by copyright or other terms. Review the official site's terms before redistributing a generated mirror.

Translations are machine-generated and unofficial. For any information with legal, financial, medical, or safety implications, verify the original source before acting.
